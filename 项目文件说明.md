# CorrTwin-Net 项目文件说明

当前项目状态：`development preview`。本轮已按照 `CorrTwin-Net_完整整改实施方案_20260606.md` 完成核心物理定义整改，但长时间 formal pipeline 和 realization 收敛实验仍未全部完成，因此不要把它描述成最终 v1.0 正式发布。

## 1. 项目当前做了什么

CorrTwin-Net 学习下面的映射：

```text
clean channel 场景参数 -> normalized ensemble |ACF| / |CCF| / |FCF|
```

正式标签不加入 AWGN，不使用 SNR 作为输入。标签通过多个独立信道 realization 的复相关 sufficient statistics 聚合得到，最后再取幅值，也就是实现 `|E[R]|`，不是 `E[|R|]`。

## 3. 关键目录

| 路径 | 说明 |
|---|---|
| `channel_sim/` | Rayleigh/Rician 信道、PDP、Rician 功率、相关函数 sufficient statistics、数据生成。 |
| `models/` | MLP、ResMLP 和 sklearn baseline。 |
| `experiments/` | 数据生成、训练、评估、理论验证、OOD、速度对比脚本。 |
| `utils/` | 归一化、指标、绘图、IO、seed 工具。 |
| `tests/` | 物理和工程测试，当前 22 个测试通过。 |
| `scripts/` | artifact manifest 生成和哈希校验脚本。 |
| `data_formal_p0/` | 本轮 formal 整改数据集。 |
| `results_formal_p0/` | 本轮 formal 指标、图、checkpoint、baseline pkl。 |
| `artifacts_manifest_formal_p0.generated.json` | 本轮 formal artifact SHA256 清单。 |
| `docs/` | 项目报告、复现说明、上传交接说明等。 |
