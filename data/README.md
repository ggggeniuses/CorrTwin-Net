# Data Directory

Generated datasets are stored here as:

- `train.npz`
- `val.npz`
- `test.npz`

These files are generated by:

```bash
python experiments/run_generate_dataset.py
```

`sample_dataset.npz` is a tiny checked-in example with 12 samples and 32 curve points for quick inspection.

The corrected generator stores metadata in `formal_dataset_metadata.json`, including feature names, label definition, and the number of realizations used for label averaging.

Generated datasets can be kept in the local release package when complete artifacts are desired. For a normal Git repository, they can also be regenerated locally instead of committed directly.
